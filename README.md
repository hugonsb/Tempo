# Tempo
Aplicativo que mostra a previsão do tempo no local exato que o usuário está.
API utilizada [(Open-Meteo)](https://open-meteo.com/).

## Tecnologias
- Framework Jetpack Compose
- Android nativo
- Retrofit
- Livedata
- Pattern MVVM
- SplashScreen
